# MBK

Quantized live record + loop machine.

Part of the connected MATTBEAR ecosystem.

## Quick Start

1. Tap WAKE
2. Tap REC on a pad
3. Wait for the grid
4. Make sound
5. Tap REC again
6. Loop starts automatically

## Pad Controls

- REC = start/stop recording
- PLAY = launch loop
- Tap looping pad = mute/unmute

## Core Rules

```text
ONE LOOP PER PAD
NO DOUBLE LOADING
QUANTIZE DEFAULT
RECORDING REPLACES DEMO LOOP
```

## Mobile Philosophy

MBK is mobile-first and designed for:

- fast live capture
- immediate interaction
- low-scroll usability
- touch-first performance
- quick experimentation
