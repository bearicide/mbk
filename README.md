# MBK

Quantized live record + loop machine.

## Quick Start

1. Tap WAKE
2. Tap REC on a pad
3. Wait for the grid
4. Make sound
5. Tap REC again
6. Loop starts automatically

## Pad Controls

- REC = start/stop recording
- PLAY = launch loop
- Tap looping pad = mute/unmute

MBK is mobile-first and designed for fast live capture instead of sample-pack browsing.
