# MBK Visual Sources

## Approved direction

- Bright electric blueprint energy
- Technical stencil fonts
- CRT glow accents
- Cyber-diner / hardware-lab vibe
- Mobile readability first

## UI direction

- Large touch targets
- Instant visual feedback
- Active pad glow states
- Recording state clarity
- Minimal menu clutter

## Avoid

- Tiny desktop-only controls
- Generic casino aesthetics
- Overloaded menus
- Long setup workflows
- Fake skeuomorphic hardware clutter

- Drum machine immediacy
- Blueprint overlays
- Neon utility panels
- Industrial music hardware
- Live performance tension
