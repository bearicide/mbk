# MBK Audio Sources

## Core starter philosophy

Default sounds should:

- Trigger instantly
- Loop cleanly
- Survive mobile speakers
- Avoid clipping stacks
- Feel playable without setup

## Suggested starter categories

- Hat loops
- Kick foundations
- Perc taps
- Bass stabs
- Vox one-shots
- Texture drones
- Noise FX
- Transition hits

## Loop behavior

- Quantized launch by default
- Recording replaces default loop on that pad
- Global choke support available
- Avoid double-loading loops already active

## Mobile-first audio notes

- Short load times
- Small preview assets
- Pre-normalized volume
- Fast attack samples
- Avoid giant stereo files unless optional
